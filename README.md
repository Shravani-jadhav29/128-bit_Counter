# 127-b-Counter
The 127-bit Counter is a Verilog HDL design implemented on the Basys 3 FPGA. It increments its 127-bit count by one on every rising edge of the 100 MHz system clock and resets to zero when the reset signal is asserted. Since the Basys 3 has only 16 LEDs, the lower 16 bits of the counter are displayed on the LEDs
